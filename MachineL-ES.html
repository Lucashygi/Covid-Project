Machine Learning in Espírito Santo Deaths

Esta parte é responsável por apresentar a evolução dos resultados de alguns algoritmos de machine learning baseados nos dados do estado do
Espírito Santo (Brasil).
Apresentando seleções de dados até Março, Abril, Maio e até o dia 19/06/2020.
Utilizando apenas valores referentes a condições biologicas e sintomáticas sendo elas:
  • Age
  • Sex
  • Ethnicity
  • Fever
  • Shortness of breath (Dyspnea)
  • Cough
  • Coryza
  • Sore Throat
  • Diarrhea
  • Headache
  • Lung Diseases
  • Cardio Diseases
  • Kidney Diseases
  • Diabetes Disease
  • Smoking
  • Obesity Disease
  • Confirmed Death
Os resultados obtidos foram através de algoritmos e filtros aplicados na ferramenta WEKA, sendo os filtros:
  • No Filtered
  • Class Balancer
  • SMOTE
No Filtered -> Sem aplicação de filtros.
Class Balancer -> Equilibra o peso das classes, tornando-os iguais.
SMOTE -> Gera valores da classe minoritária baseados nos valores próximos. (Todos os casos com SMOTE o filtro foi aplicado deixando as classes iguais).

Os algoritmos utilizados foram os seguintes:
  • Decision Table
  • Logistic
  • Naïve Bayes
  • Simple Logistic
  • SPegasos
  • Support Vector Machine (SVM)
  • REP Tree

SPegasos and SVM Don't have implementation using Class Balancer and Without a Cost Matrix
