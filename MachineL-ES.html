Machine Learning in Espírito Santo Deaths

This part is responsable for presenting the evolution in some results of Machine Learning algorithms based in the state of Espírito Santo (Brasil) datasets.

Presenting data selection until March, April, May and until the day June 16, 2020.
Using only values referring to biological and symptomatic conditions as presented below:

  • Age
  • Sex
  • Ethnicity
  • Fever
  • Shortness of breath (Dyspnea)
  • Cough
  • Coryza
  • Sore Throat
  • Diarrhea
  • Headache
  • Lung Diseases
  • Cardio Diseases
  • Kidney Diseases
  • Diabetes Disease
  • Smoking
  • Obesity Disease
  • Confirmed Death

The results were found through algorithms and filters applied in WEKA Workbench.
The filter were:

  • No Filtered
  • Class Balancer
  • SMOTE

No filtered -> With no filters.
Class Balancer -> It balances the weight of the classes, turning them equal.
SMOTE -> Build values in the lower class based in neighboors values. (Every cases using SMOTE the filter was applied setting both classes equal).

The algorithms used were the following:

  • Decision Table
  • Logistic
  • Naïve Bayes
  • Simple Logistic
  • SPegasos
  • Support Vector Machine (SVM)
  • REP Tree

SPegasos and SVM Don't have implementation using Class Balancer Without a Cost Matrix
